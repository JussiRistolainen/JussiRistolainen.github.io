TideMesh — Documentation

Requirements

Unity 2021.3 LTS or newer • URP/HDRP.

Install

  1. Import the package.
  2. Open the sample scene.
  3. Press Play to verify.

Quick Start

  1. Add TideMesh to your scene.
  2. Assign your water material.
  3. Drop in boats/buoys.
  4. Tweak wave scale, foam, color.
  5. Build & test.
